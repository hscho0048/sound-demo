# GLB 에셋 배치 안내

이 폴더는 Three.js 3D 홈 화면에서 사용할 GLB 파일을 배치하는 위치입니다.

MVP 저장소에는 대용량 또는 저작권이 있는 3D 모델을 포함하지 않습니다. 실제 모델 파일은 개발자가 직접 준비해 아래 이름으로 추가합니다.

```text
robot_vacuum.glb
home_structure.glb
washing_machine.glb
dishwasher.glb
smartphone_device.glb
```

파일이 없을 때 앱은 기본 도형으로 대체 표시합니다. 로봇청소기 경로 변경은 실제 기기 제어가 아니라 프론트엔드 시뮬레이션입니다.
